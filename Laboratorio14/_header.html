Cuestionario

¿Qué es ODBC y para qué es útil?

Permite acceder a una base de datos sin importar la interfaz. Es decir, poder conectarse a una misma base de datos con diferentes aplicaciones.

¿Qué es SQL Injection?

Al igual que los otros tipos de ataques que hemos visto en otros laboratorios, este ataque se realiza cuando un usuario ingresa un comando como un atributo. En este caso, ocurre cuando el usuario ingresa un comando SQL para ingresar, modificar, borrar, y alterar la base de datos.

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Una tecnica util para evitar estos ataques es convertir el ingreso de datos al tipo de datos que se requiere. Otra tecnica es evitar caracteres especiales para evitar comandos especiales tipo REGEX. Por ultimo, se puede restringir el acceso al usuario para que no pueda hacer alteraciones drasticas a la base de datos.
