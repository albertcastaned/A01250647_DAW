¿Qué es entonces una base de datos - DB?

Es un conjunto de volúmenes de datos relacionados que requieren ser estructurados con alguna organización para poder ser consultados. [1]

¿En qué casos es conveniente usar bases de datos?

En casos que es necesario tener un sistema de almacenamiento escalable, rápido, y seguro. Cuando hay datos que pueden tener relación entre si, es un indicador que una base de datos puede ser mas conveniente.

¿Qué es un sistema de gestión de base de datos - DBMS?

DBMS o Data Base Management System es un software que controla la organización, almacenamiento, recuperación, seguridad, e integridad de los datos en una base de datos. [2]

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  • Crear, manipular, y organizar bases de datos de acuerdo a las necesidades.
  • Contar con mecanismos que permitan la recuperación de datos si llega a pasar fallas.
  • Control de concurrencia para no afectar la inconsistencia de datos.
