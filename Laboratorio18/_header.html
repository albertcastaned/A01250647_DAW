Cuestionario

¿Qué importancia tiene AJAX en el desarrollo de RIA's Rich Internet Applications?

La implementacion de AJAX reduce el tiempo de respuesta de una peticion por el navegodar web, lo que mejora la experiencia RIA y hace que estas aplicaciones lleguen al nivel de una aplicacion de escritorio.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

El llamado de JS se envian por texto plano, por lo cual es posible que valores importantes como IDs de Usuarios que se utilizan en AJAX puedan ser accesibles por otros. Para evitar esto, es importante encriptar y desencriptar los datos del lado del cliente y del servidor respectivamente.

¿Qué es JSON?

Javascript Object Notation es un formato de texto sencillo y legible tanto para el humano como para computadoras. Se utiliza en la modernidad para recibir y enviar datos en la web ya que es mas facil obtener sus atributos que otros formatos como XML.
